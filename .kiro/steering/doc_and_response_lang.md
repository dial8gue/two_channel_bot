# Языковые требования

## Основной язык проекта

Все взаимодействие с пользователем должно происходить на **русском языке**:

- Ответы и объяснения ИИ-ассистента
- Документация
- Сообщения об ошибках

## Исключения

Английский язык допустим только для:
- Названий переменных, функций и классов в коде
- Технических терминов без устоявшегося русского перевода
- Внешних API и библиотек
- Конфигурационных файлов
- Комментарии к коду
- Логи и отладочная информация

## Стиль общения

- Используйте профессиональный, но дружелюбный тон
- Объясняйте технические концепции простым языком
- При необходимости приводите примеры на русском языке
- Избегайте излишнего использования англицизмов
